# ポモドーロタイマー Webアプリ - 仕様書

## 📋 プロジェクト概要

ポモドーロテクニックに基づいた生産性向上ツール。液体の視覚効果を取り入れた美しいWebアプリケーション。YouTube音楽統合により、作業中に好きな音楽を再生できます。

---

## 🎯 主要機能

### 1. タイマー機能

#### フェーズ構成
- **作業フェーズ**: 25分 × 4セット
- **短い休憩フェーズ**: 5分 × 4回（各作業後）
- **長い休憩フェーズ**: 30分（4セット完了後）

#### タイマー動作
| フェーズ | 時間 | 回数 | スケジュール |
|---------|------|------|------------|
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| **長い休憩** | **30分** | **1回** | **(サイクル完了)** |

#### タイマー操作
- **開始**: タイマーを開始し、カウントダウン開始
  - ボタン状態: 「開始」ボタン無効化 → 「一時停止」ボタン有効化
  - 自動遷移: 各フェーズ完了時に自動的に次のフェーズへ移行
- **一時停止**: タイマーを一時停止
  - ボタン状態: 「一時停止」ボタン無効化 → 「開始」ボタン有効化
  - 時間は保持される
- **リセット**: タイマーを最初（作業フェーズ25分、セット1/4）に戻す
  - 音楽再生も停止する

## 🎨 UI/UX設計

### 液体アニメーション
- **概要**: 画面全体が液体で満たされており、時間経過に応じて液体レベルが低下
- **開始時**: 液体が画面全体を100%満たす
- **進行中**: 経過時間に応じた比率で液体が減少
  - 残り時間 / 総時間 = 液体充填率
- **完了時**: 液体が0%（画面が空になる）
- **波のアニメーション**: 液体表面に波のエフェクト（8秒周期でループ）
- **色**: グラデーション（上部: #ff6b6b → 下部: #ee5a6f）

### タイマー表示

#### 時刻表示
- **形式**: MM:SS（分:秒）
- **フォントサイズ**: 120px（デスクトップ）、80px（モバイル）
- **フォントウェイト**: Light (300)
- **表示位置**: 画面中央上部
- **影**: テキストシャドウで視認性確保

#### プログレスリング
- **形式**: 円形プログレスバー（時刻表示の下部）
- **サイズ**: 直径200px
- **背景**: 半透明白（rgba(255, 255, 255, 0.3)）
- **進捗バー**: 白色、ストローク幅4px
- **動作**: 時間経過に応じてリング満度が増加

### フェーズ表示

#### フェーズテキスト
- **表示位置**: ヘッダー右側
- **背景**: 半透明ガラス効果（backdrop-filter: blur(10px)）
- **フェーズ別表示**:
  - 作業中: 「作業中」(赤系背景)
  - 短い休憩中: 「短い休憩中」(緑系背景)
  - 長い休憩中: 「長い休憩中」(青系背景)

#### セットカウンター
- **表示位置**: フェーズテキストの左側
- **表示形式**: 「セット X/4」
- **更新**: セット切り替え時にリアルタイム更新
- **完了時**: 「サイクル完了！」に変更

### コントロールボタン

#### ボタン一覧
| ボタン | 通常状態 | 有効/無効 | 機能 |
|--------|---------|---------|------|
| 開始 | 背景白/テキスト赤 | タイマー停止中のみ有効 | タイマー開始 |
| 一時停止 | 背景半透明白/テキスト白 | タイマー実行中のみ有効 | タイマー一時停止 |
| リセット | 背景半透明白/テキスト白 | 常に有効 | 全設定をリセット |

#### ボタンのスタイル
- **パディング**: 12px 32px
- **フォントサイズ**: 16px
- **ボーダーラディウス**: 25px（ピル型）
- **ホバー効果**: スケール(1.05) + シャドウ追加
- **トランジション**: 0.3秒

## 🎵 YouTube音楽統合

### 概要
ユーザーがYouTube動画のURLを入力するだけで、その動画の音声をバックグラウンドで再生できる機能。

### 入力フォーム
- **配置**: 画面下部（固定）
- **背景**: 半透明黒（rgba(0, 0, 0, 0.5)）
- **ガラス効果**: blur(10px)
- **レイアウト**: 横並び（入力フィールド + 設定ボタン）

### 対応URL形式
1. **長いURL**: `https://www.youtube.com/watch?v=xxxxx`
2. **短いURL**: `https://youtu.be/xxxxx`

### 動作仕様
- **入力フィールド**:
  - プレースホルダー: 「YouTubeの動画URLを貼り付け (例: https://www.youtube.com/watch?v=...)」
  - 最大幅: 600px（デスクトップ）
  
- **設定ボタン**:
  - クリックで動画IDを抽出
  - 無効なURLの場合: アラート表示「有効なYouTube動画URLを入力してください」
  - Enterキーでも設定可能（UX向上）

- **プレーヤー表示**:
  - 動画が設定されると、YouTubeプレーヤーが表示
  - アスペクト比: 16:9
  - ボーダーラディウス: 8px
  - 初期状態: 非表示（動画設定時に表示）
  - 自動再生: フォーカス時に自動再生開始

## 📊 状態管理

### タイマー状態オブジェクト
```
timerState = {
  isRunning: boolean,           // タイマー実行中フラグ
  totalSeconds: number,         // 現在のフェーズの総時間（秒）
  remainingSeconds: number,     // 残り時間（秒）
  currentPhase: string,         // 'work' | 'shortBreak' | 'longBreak'
  currentSet: number,           // 現在のセット番号 (1-4)
  totalPhases: number,          // 全フェーズを通じての進捗
  intervalId: number            // setInterval ID
}
```

### 状態遷移図
```
開始時
  ↓
作業フェーズ (25分)
  ↓ 完了
短い休憩 (5分) - セット1
  ↓ 完了
作業フェーズ (25分)
  ↓ 完了
短い休憩 (5分) - セット2
  ↓ 完了
[... セット3, 4 も同様 ...]
  ↓ 完了
長い休憩 (30分)
  ↓ 完了
サイクル完了 → リセット
```

## ⚡ パフォーマンス考慮

- **アニメーション**: GPU加速対応（transform, opacity使用）
- **再描画**: 必要最小限に抑える
- **タイマー精度**: setInterval（1000ms周期）
- **メモリリーク**: タイマー停止時にintervalを確実に削除
