# ポモドーロタイマー Webアプリ - 詳細仕様書

ポモドーロテクニックに基づいた生産性向上ツール。液体の視覚効果を取り入れた美しいWebアプリケーション。YouTube音楽統合により、作業中に好きな音楽を再生できます。

---

## 🎯 主要機能

### 1. タイマー機能

#### フェーズ構成
- **作業フェーズ**: 25分 × 4セット
- **短い休憩フェーズ**: 5分 × 4回（各作業後）
- **長い休憩フェーズ**: 30分（4セット完了後）

#### タイマー動作
| フェーズ | 時間 | 回数 | スケジュール |
|---------|------|------|------------|
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| **長い休憩** | **30分** | **1回** | **(サイクル完了)** |

#### タイマー操作
- **開始**: タイマーを開始し、カウントダウン開始
  - ボタン状態: 「開始」ボタン無効化 → 「停止」ボタン有効化
  - 自動遷移: 各フェーズ完了時に自動的に次のフェーズへ移行
  - YouTube動画の自動再生: 開始時に設定済みの動画が自動再生開始
- **停止**: タイマーを一時停止
  - ボタン状態: 「停止」ボタン無効化 → 「開始」ボタン有効化
  - 時間は保持される
  - YouTube動画も一時停止される
- **リセット**: タイマーを最初（作業フェーズ25分、セット1/4）に戻す
  - 音楽再生も停止する

## 🎨 UI/UX設計

### 液体アニメーション
- **概要**: 画面全体が液体で満たされており、時間経過に応じて液体レベルが低下
- **開始時**: 液体が画面全体を100%満たす
- **進行中**: 経過時間に応じた比率で液体が減少
  - 残り時間 / 総時間 = 液体充填率
- **完了時**: 液体が0%（画面が空になる）
- **波のアニメーション**: 液体表面に波のエフェクト（8秒周期でループ）
- **色**: グラデーション
  - 上部: rgba(107, 154, 255, 0.85)
  - 中部: rgba(90, 143, 255, 0.90) / rgba(65, 105, 255, 0.95)
  - 下部: rgba(45, 95, 255, 1)

### タイマー表示

#### 時刻表示
- **形式**: MM:SS（分:秒）
- **フォントサイズ**: 
  - デスクトップ: 120px
  - モバイル（768px未満）: 80px
- **フォント**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
- **表示位置**: 画面中央
- **スタイル**: テキストシャドウで視認性確保
- **カスタマイズ可能**: 設定パネルでフォントサイズ・フォント・太さを変更可能

### フェーズ表示

#### フェーズテキスト＆セットカウンター
- **表示位置**: ヘッダー右側
- **背景**: 半透明ガラス効果（backdrop-filter: blur(10px)）
- **フェーズ別表示**:
  - 作業中: 「work」
  - 短い休憩中: 「short break」
  - 長い休憩中: 「long break」
- **セット表示**: 「set 1/4」などの形式でセット進捗を表示
- **更新**: リアルタイムで更新

### コントロールボタン

#### ボタン一覧
| ボタン | 状態 | 機能 |
|--------|------|------|
| start | 停止中のみ有効 | タイマー開始 |
| stop | 実行中のみ有効 | タイマー一時停止 |
| reset | 常に有効 | 全設定をリセット |

#### ボタンのスタイル
- **パディング**: 12px 32px
- **フォントサイズ**: 16px
- **ボーダーラディウス**: 25px（ピル型）
- **primary**: 白背景/赤テキスト
- **secondary**: 半透明白背景/白テキスト
- **ホバー効果**: スケール(1.05) + シャドウ追加
- **トランジション**: 0.3秒

## 🎵 YouTube音楽統合

### 概要
ユーザーがYouTube動画のURLを入力するだけで、その動画の音声をバックグラウンドで再生できる機能。【設定パネルで実装】

### 対応URL形式
1. **長いURL**: `https://www.youtube.com/watch?v=xxxxx`
2. **短いURL**: `https://youtu.be/xxxxx`

### 動作仕様
- **動画IDの自動抽出**: URLから動画IDを抽出して処理
- **無効なURL**: アラート表示「有効なYouTube動画URLを入力してください」
- **自動再生**: 
  - タイマー開始時に設定済みの動画が自動再生開始
  - フォーカス時に自動再生開始
- **再生制御**:
  - タイマー一時停止時に動画も一時停止
  - タイマー再開時に動画も再開
  - リセット時に動画も停止

## 📊 状態管理

### タイマー状態オブジェクト
```javascript
timerState = {
  isRunning: boolean,           // タイマー実行中フラグ
  totalSeconds: number,         // 現在のフェーズの総時間（秒）
  remainingSeconds: number,     // 残り時間（秒）
  currentPhase: string,         // 'work' | 'shortBreak' | 'longBreak'
  currentSet: number,           // 現在のセット番号 (1-4)
  totalPhases: number,          // 全フェーズを通じての進捗 (0-11: work+break, 12: long break)
  intervalId: number,           // setInterval ID
  audioContext: object,         // Audio API コンテキスト
  player: object                // YouTube プレーヤー
}
```

### 状態遷移図
```
初期状態（作業フェーズ 25分）
  ↓ [開始]
作業フェーズ (25分) - セット1
  ↓ [完了]
短い休憩 (5分) - セット1
  ↓ [完了]
作業フェーズ (25分) - セット2
  ↓ [完了]
短い休憩 (5分) - セット2
  ↓ [完了]
[... セット3, 4 も同様 ...]
  ↓ [完了]
長い休憩 (30分)
  ↓ [完了]
サイクル完了 → 初期状態にリセット
```

## 📋 使い方

1. `index.html`をブラウザで開きます
2. 「start」ボタンをクリックしてタイマーを開始
3. 右下の設定アイコン（⚙️）をクリックして、YouTube動画URLを貼り付け
4. タイマーは自動で進行し、フェーズが完了すると自動で次へ移行します
5. 「stop」ボタンで一時停止、「reset」ボタンでリセット可能
