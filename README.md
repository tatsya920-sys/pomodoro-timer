# ポモドーロタイマー Webアプリ - 詳細仕様書

ポモドーロテクニックに基づいた生産性向上ツール。液体の視覚効果を取り入れた美しいWebアプリケーション。YouTube音楽統合により、作業中に好きな音楽を再生できます。

---

## 🎯 主要機能

### 1. タイマー機能

#### フェーズ構成
- **作業フェーズ**: Nセット（N = 設定パネルで自由に変更可能、デフォルト: 25分）
- **短い休憩フェーズ**: Nセット（各作業後、デフォルト: 5分）
- **長い休憩フェーズ**: 1回（Nセット完了後、デフォルト: 30分）
- **自動継続**: 長い休憩終了後、自動でセット数がリセットされて、次のサイクルが開始される

*注: N、休憩時間、長い休憩時間はすべてカスタマイズ可能です*

#### タイマー動作
| フェーズ | 時間 | 回数 | スケジュール |
|---------|------|------|------------|
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| 作業 | 25分 | 1回 | |
| 短い休憩 | 5分 | 1回 | |
| **長い休憩** | **30分** | **1回** | **(サイクル完了)** |

#### タイマー操作
- **開始**: タイマーを開始し、カウントダウン開始
  - ボタン状態: 「開始」ボタン無効化 → 「停止」ボタン有効化
  - フェーズ移行時は一時停止状態になり、再度STARTボタンの押下が必要
- **停止**: タイマーを一時停止
  - ボタン状態: 「停止」ボタン無効化 → 「開始」ボタン有効化
  - 時間は保持される
- **リセット**: タイマーを最初（作業フェーズ25分、セット1/N、Nは設定パネルで指定した値）に戻す
  - サイクルカウントと設定もリセット

#### フェーズ完了通知
- **音声フィードバック**: 各フェーズ完了時に3段階の上昇するビープ音を再生
  - 周波数: 800Hz → 1000Hz → 1200Hz
  - 継続時間: 各0.15秒、間隔0.1秒
  - ユーザーが現在のフェーズ完了に気付きやすくなります
- **アラート**: 長い休憩完了（サイクル完了）時に完了メッセージを表示

## 🎨 UI/UX設計

### 液体アニメーション
- **概要**: 画面全体が液体で満たされており、時間経過に応じて液体レベルが低下
- **開始時**: 液体が画面全体を100%満たす
- **進行中**: 経過時間に応じた比率で液体が減少
  - 残り時間 / 総時間 = 液体充填率
- **完了時**: 液体が0%（画面が空になる）
- **波のアニメーション**: 液体表面に波のエフェクト（8秒周期でループ）
- **色**: グラデーション
  - 上部: rgba(107, 154, 255, 0.85)
  - 中部: rgba(90, 143, 255, 0.90) / rgba(65, 105, 255, 0.95)
  - 下部: rgba(45, 95, 255, 1)

### タイマー表示

#### 時刻表示
- **形式**: MM:SS（分:秒）
- **フォントサイズ**: 180px（固定）
- **フォント**: Segoe UI（固定）
- **表示位置**: 画面中央
- **スタイル**: テキストシャドウで視認性確保

### フェーズ表示

#### フェーズテキスト＆セットカウンター＆サイクルカウンター
- **表示位置**: ヘッダー右側
- **背景**: 半透明ガラス効果（backdrop-filter: blur(10px)）
- **フェーズ別表示**:
  - 作業中: 「work」
  - 短い休憩中: 「short break」
  - 長い休憩中: 「long break」
- **セット表示**: 「set 1/4」などの形式でセット進捗を表示
  - 長い休憩中: 「set - / 4」のように表示
- **サイクル表示**: 「cycle 0」などの形式で完了したサイクル数を表示
  - 1サイクル（Nセット + 長い休憩）終了するとカウントが増える
- **更新**: リアルタイムで更新

### コントロールボタン

#### ボタン一覧
| ボタン | 状態 | 機能 |
|--------|------|------|
| start | 停止中のみ有効 | タイマー開始 |
| stop | 実行中のみ有効 | タイマー一時停止 |
| reset | 常に有効 | 全設定をリセット |

#### ボタンのスタイル
- **パディング**: 12px 32px
- **フォントサイズ**: 16px
- **ボーダーラディウス**: 25px（ピル型）
- **全ボタン**: 白背景/青（#2d5fff）テキスト
  - start、stop、reset 全て同じスタイルで統一
- **ホバー効果**: スケール(1.05) + シャドウ追加
- **トランジション**: 0.3秒

## 🎵 YouTube音楽統合

### 概要
ユーザーがYouTube動画のURLを入力するだけで、その動画の音声をバックグラウンドで再生できる機能。【設定パネルで実装】

### 対応URL形式
1. **長いURL**: `https://www.youtube.com/watch?v=xxxxx`
2. **短いURL**: `https://youtu.be/xxxxx`

### 動作仕様
- **動画IDの自動抽出**: URLから動画IDを抽出して処理
- **無効なURL**: アラート表示「有効なYouTube動画URLを入力してください」
- **自動再生**: 
  - タイマー開始時に設定済みの動画が自動再生開始
  - フォーカス時に自動再生開始
- **再生制御**:
  - タイマー一時停止時に動画も一時停止
  - タイマー再開時に動画も再開
  - リセット時に動画も停止

## ⚙️ 設定パネル

### カスタマイズ機能

#### フォント表示設定
- **フォントサイズ**: 180px（固定）
- **フォント**: Segoe UI（固定）
- **フォント太さ**: Bold 700（固定）
- **テキスト色**: #ffffff（白、固定）

#### タイマー設定
- **セット数**: 1～20セット の範囲で自由に変更（デフォルト: 4セット）
  - 指定したセット数分の作業 → 短い休憩を繰り返した後、長い休憩が実行されます
  - タイマー実行中は変更できません
- **作業時間**: 1～60分 の範囲で自由に変更（デフォルト: 25分）
  - タイマー実行中は変更できません
- **短い休憩時間**: 1～30分 の範囲で自由に変更（デフォルト: 5分）
  - タイマー実行中は変更できません
- **長い休憩時間**: 1～60分 の範囲で自由に変更（デフォルト: 30分）
  - タイマー実行中は変更できません

### 配置
- **トグルボタン**: 画面右下に浮かぶ（⚙️ アイコン）
- **パネル**: トグル時に表示/非表示切り替え

## 📊 状態管理

### タイマー状態オブジェクト
```javascript
timerState = {
  isRunning: boolean,           // タイマー実行中フラグ
  totalSeconds: number,         // 現在のフェーズの総時間（秒）
  remainingSeconds: number,     // 残り時間（秒）
  currentPhase: string,         // 'work' | 'shortBreak' | 'longBreak'
  currentSet: number,           // 現在のセット番号（1～maxSets）
  maxSets: number,              // 長い休憩までのセット数（デフォルト: 4）
  cycleCount: number,           // 完了したサイクル数
  totalPhases: number,          // 全フェーズを通じての進捗
  intervalId: number,           // setInterval ID
  audioContext: object,         // Audio API コンテキスト
  player: object                // YouTube プレーヤー
}
```

### 状態遷移図
```
初期状態（cycle: 0, set: 1, work フェーズ）
  ↓ [開始]
work (25分) - set 1
  ↓ [完了]
short break (5分) - set 1
  ↓ [完了]
work (25分) - set 2
  ↓ [完了]
short break (5分) - set 2
  ↓ [完了]
[... set 3, 4 も同様 ...]
  ↓ [完了]
long break (30分)
  ↓ [完了]
work (25分) - set 1【cycleカウント = 1で自動継続】
  ↓
[... 無限ループ継続 ...]
```

## 📋 使い方

1. `index.html`をブラウザで開きます
2. 「start」ボタンをクリックしてタイマーを開始
3. 右下の設定アイコン（⚙️）をクリックで、セット数・作業時間・休憩時間を設定
4. タイマーは自動で進行し、フェーズが完了すると自動で次へ移行します
5. 長い休憩が終わると、自動で次のサイクルが開始されます（cycle カウントが増加）
6. 「stop」ボタンで一時停止、「reset」ボタンでリセット可能（cycle と set がリセット）
