# ポモドーロタイマー Webアプリ

美しい液体アニメーションと YouTube 音楽統合を備えたポモドーロテクニックタイマー。

## 🎯 機能

- ✨ **液体アニメーション**: 時間経過に応じて液体レベルが低下
- ⏱️ **ポモドーロタイマー**: 25分×4セット + 5分休憩 + 30分大休憩
- 🎵 **YouTube音楽統合**: URLを貼るだけで音楽を再生
- 📱 **レスポンシブデザイン**: モバイル・タブレット・デスクトップ対応
- 🎨 **美しいUI**: モダンでミニマルなデザイン

## 📋 使い方

1. `index.html`をブラウザで開きます
2. 「開始」ボタンをクリックしてタイマーを開始
3. YouTube動画URLを入力して「設定」をクリックすると音楽が再生されます
